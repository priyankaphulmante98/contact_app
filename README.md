## Run Locally

Clone the project

```bash
 https://github.com/priyankaphulmante98/contact_app.git
```

Go to the project directory

```bash
  cd contact_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
